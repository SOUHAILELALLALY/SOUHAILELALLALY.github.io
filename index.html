AI Assistant for Car Brand Identification

Developed an interactive web application for car brand recognition by fine-tuning a pre-trained ResNet50 model for image classification. Implemented image similarity retrieval using deep features to find visually similar cars. Integrated a Large Language Model (LLM) to serve as an AI assistant, providing users with detailed information about the identified car brands. The solution was deployed via a user-friendly Streamlit interface, enabling image uploads, brand classification, and similarity search in real time.

PDF Chatbot

Developed an interactive web application that enables users to upload PDF documents and engage in conversational question-answering based on the document content. Leveraged Google’s Gemini large language model for natural language understanding, LangChain for text chunking and QA pipeline orchestration, and FAISS for efficient semantic vector search. Utilized Streamlit to build a user-friendly interface allowing seamless PDF upload, text processing, and dynamic querying. Implemented end-to-end PDF text extraction, embedding generation, and context-aware response generation to provide accurate and detailed answers grounded in the uploaded documents.

LLMs Streamlit LangChain Python Generative AI

Shortest Path Optimization with Dijkstra’s Algorithm

Implemented Dijkstra’s algorithm to compute the shortest path from a source node to all other nodes in a weighted graph. The project focused on efficient distance updates, node selection, and graph traversal to handle non-negative edge weights. This classic optimization technique is foundational in network routing, GPS navigation, and AI pathfinding applications, and was applied to demonstrate practical use cases in graph-based problem solving.
