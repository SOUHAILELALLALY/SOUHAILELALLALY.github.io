Multimodal Car Analysis

Developed an AI-powered web application that enables users to upload car images for brand classification, visual similarity retrieval, and detailed insights. Leveraging Streamlit for the interface, PyTorch and CLIP for image embeddings, and FAISS for efficient similarity search using either images or text descriptions, the platform delivers fast and accurate results. Integrated the Google Gemini LLM via LangChain to generate intelligent descriptions and answer user queries about the uploaded car. Designed an interactive chat interface that provides real-time, context-aware responses, creating a seamless multimodal car-analysis experience.

PyTorch Scikit-learn Streamlit LLMs

Financial Fraud Detection

This project focuses on building a fraud detection system for financial transactions using supervised machine learning techniques. Since fraudulent transactions are rare compared to legitimate ones, the dataset is highly imbalanced, making the problem more challenging. To address this, we performed data visualization, then trained and evaluated two models—Logistic Regression and Random Forest—to determine which performed best at identifying fraud. Finally, we deployed the solution as an interactive Streamlit application, allowing users to input transaction data and receive real-time predictions on whether a transaction is fraudulent.

Document AI Assistant

Built an interactive web app enabling users to upload PDFs and engage in conversational question answering grounded in the document’s content. Designed a multimodal RAG pipeline that processes both text and image data from PDFs. Integrated Google Gemini for natural language understanding, LangChain for text chunking and context management, and CLIP for generating unified text–image embeddings stored in a FAISS vector database for efficient semantic retrieval. Developed a streamlined Streamlit interface supporting PDF upload, embedding generation, and dynamic, context-aware responses.
